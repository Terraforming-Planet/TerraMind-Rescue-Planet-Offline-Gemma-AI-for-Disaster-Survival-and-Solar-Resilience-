# Uploaded Files Description

This ZIP package contains the first-day project foundation for TerraMind Rescue Planet.

## Files

### README.md
Main project README for GitHub. It explains the problem, solution, workflow, Gemma usage, demo scenarios, and social impact.

### PROJECT_DESCRIPTION.md
Clean English project description for Kaggle writeup. It includes a better title, subtitle, and full project explanation.

### DEMO_SCENARIOS.md
Four focused demo scenarios:
1. Flood rescue
2. Damaged power infrastructure
3. Fire and smoke detection
4. Solar route and PV parking assistant

### GEMMA_PROMPT.md
The main prompt for Gemma Vision. It defines how the AI should analyze images and return structured emergency output.

### JSON_SCHEMA_EXAMPLE.json
Example JSON output showing how the backend/frontend can use Gemma results.

## Purpose

These files are intended to make the project look organized, clear, and realistic from day one. They can be uploaded to GitHub and used as supporting files for the Kaggle hackathon submission.
